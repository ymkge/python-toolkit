# TSV Splitter

## 概要

このツールは、大きなTSV (Tab-Separated Values) ファイルを、指定した行数ごとに複数の小さなファイルに分割します。

## 機能

- **ファイル分割**: TSVファイルを指定した行数で分割します。デフォルトは30,000行です。
- **文字数制限**: 各セルの内容は45,000文字に制限されます。これを超える文字は切り捨てられます。
- **エラーハンドリング**: 不正な行や空行があっても処理を継続します（詳細はターミナルに表示されます）。

## 使い方

### 実行コマンド

ターミナルで以下のコマンドを実行します。1ファイルあたりの分割行数はオプションで指定できます。

```bash
python3 tsv_split.py [分割したいファイル名.tsv] [オプション: 1ファイルあたりの行数]
```

### 例

#### デフォルトの30,000行で分割する場合

`input.tsv` というファイルを分割する場合：

```bash
python3 tsv_split.py input.tsv
```

#### 50,000行で分割する場合

```bash
python3 tsv_split.py input.tsv 50000
```

### 出力

実行すると、`input_split_result_1.tsv`, `input_split_result_2.tsv`, ... のような名前のファイルが生成されます。

## 要件

このスクリプトの実行には `pandas` ライブラリが必要です。以下のコマンドでインストールしてください。

```bash
pip install pandas
```
