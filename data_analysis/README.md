# データ分析 (Data Analysis)

このディレクトリには、データを分析し、知見を得るためのツールやテンプレートを格納します。

## 機能一覧

- **[データ可視化 (Data Visualization)](./visualization/README.md)**
  - CSVデータからグラフを生成し、可視化するためのツール群です。
  - `basic_plotter`: 棒グラフや散布図を生成します。（実装済み）

- **探索的データ分析 (Exploratory Data Analysis - EDA)**
  - データセットの基本的な統計量を算出したり、データの全体像を素早く把握するためのJupyter Notebookテンプレートやスクリプトです。（未実装）

- **統計分析 (Statistical Analysis)**
  - t検定やカイ二乗検定、相関分析など、データから統計的に意味のある結論を導き出すためのスクリプト群です。（未実装）
