# インフラ構築・運用 (Infrastructure)

このディレクトリには、サーバーやクラウド環境の構築、運用、自動化を支援するためのスクリリプトやツールを格納します。

## 機能一覧

- **[ログパーサー (Log Parser)](./log_parser/README.md)**
  - Webサーバー等のログファイルを解析し、情報を抽出・集計するツール群です。
  - `access_log_parser`: Apacheのアクセスログを解析します。（実装済み）

- **AWSヘルパー (AWS Helpers)**
  - `boto3`ライブラリをラップし、S3へのファイルアップロード・ダウンロードやEC2インスタンスの操作など、AWS上の一般的な操作を簡単に行うためのスクリプト群です。（未実装）

- **SSH自動化 (SSH Automation)**
  - `paramiko`や`fabric`といったライブラリを利用して、複数のリモートサーバーにSSHで接続し、コマンドの実行やファイルの配布などを自動化します。（未実装）
